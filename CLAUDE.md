# UnoLock — Claude Code 안내 (저장소 루트)

작업을 시작하기 전에 아래를 순서대로 먼저 읽으세요:

1. **`unolock-docs/CLAUDE.md`** — 프로젝트 전체 안내 (철학·사용자 구조·MVP 범위). 가장 중요.
2. **`unolock-docs/설계자료/`** — 기획서, 계정/화면/보상 다이어그램, 학생 화면 시안.
3. **`PROGRESS.md`** — 지금까지 진행 상황과 **바로 다음 할 일**.

## 핵심 작업 규칙 (대표님 요청)
- 기획자는 비전공자 → **무엇을·왜 하는지 쉽게 먼저 설명**하고 진행.
- **한 단계씩**, 매 단계 끝나면 확인받고 다음으로.
- 민감정보(API 키·비밀번호)는 **사용자가 직접 입력.** Claude는 위치만 안내하고 값은 보지 않으며, 코드·로그·대화에 노출하지 않음.
